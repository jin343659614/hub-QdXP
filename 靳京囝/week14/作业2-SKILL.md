# Skill：股票可视化与买卖建议

## 技能名称
stock_visualize_and_trade_advice

## 技能描述
获取指定股票的日K线与周K线数据，将日波动与周波动绘制在同一张图表中；
自动分析趋势、支撑位、压力位，并基于日线与周线共振给出最佳买入、卖出、观望建议。

## 输入参数
- code：股票代码（字符串，必填）
- stock_name：股票名称（字符串，可选，默认“股票”）

## 输出内容
1. 日K线 + 周K线叠加可视化图表
2. 趋势判断（上升/下降/震荡）
3. 支撑位与压力位
4. 最佳买卖时间建议
5. 完整分析报告

## 依赖接口
- https://api.autostock.cn/v1/stock/kline/day （日K线）
- https://api.autostock.cn/v1/stock/kline/week （周K线）

## 分析规则
1. 周线20日均线为主要支撑/趋势判断线
2. 周线向上 + 日线企稳 → 买入/加仓
3. 周线向下 + 日线走弱 → 卖出/空仓
4. 横盘震荡 → 观望或小仓位波段
